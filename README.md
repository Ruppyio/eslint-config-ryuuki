## Description
This config extend  airbnb's config with my personal preferences.

## Usage
Install with `npm install -D eslint-config-ryuuki`.

Create `.eslintrc.json` file in root folder:
```json
{
  "extends": "ryuuki"
}
```
or `.eslintrc.yml` file:
```yml
extends: ryuuki
```
